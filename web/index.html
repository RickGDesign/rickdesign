Aplicación Firma Ec

Diseñé el nuevo logotipo para la aplicación lanzada por el gobierno para las firmas electrónicas, además de realizar la propuesta de diseño de interfaz para la app, realicé el logotipo y logo para el lanzador, cromática, tipografía y los iconos.

Cromática

Se escogieron tonalidades acordes al antiguo logotipo pero que tengan armonía con la aplicación y fondos de la misma.

Tipografía

Se usaron tipografías legibles y versátiles para todo tipo de usuario que descargue la aplicación.

Logotipos

Se presentan las 2 propuestas nuevas del logotipo para ser lanzadas a las tiendas de dispositivos inteligentes y ser reconocidas con mayor facilidad.

Iconos

Diseño de iconos que van a ir acorde a lo que el usuario requiera junto a una breve descripción.
